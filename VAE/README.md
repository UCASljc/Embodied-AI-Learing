# VAE（变分自编码器）学习与实践

## 学习目标
- 理解VAE的基本原理与数学推导
- 掌握VAE的PyTorch实现方法
- 通过实验掌握VAE在图像生成等任务中的应用

## 理论要点
- 概率图模型与生成模型基础
- 变分推断与ELBO（证据下界）
- 编码器与解码器结构
- 重参数化技巧（Reparameterization Trick）

## 推荐资料
- Kingma, D.P. & Welling, M. (2014). Auto-Encoding Variational Bayes. [arXiv:1312.6114](https://arxiv.org/abs/1312.6114)
- PyTorch官方教程：[VAE Example](https://github.com/pytorch/examples/blob/main/vae/main.py)
- 李沐《动手学深度学习》VAE章节
